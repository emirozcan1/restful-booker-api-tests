# Restful Booker API Test Project

This project includes API tests for the Restful Booker application.

## Tested Endpoints
- Authentication (Create Token)
- Get Booking IDs
- Create Booking
- Get Booking Details
- Update Booking
- Partial Update Booking
- Delete Booking
- Health Check

## Tools & Technologies
- Postman
- REST API
- JSON
- HTTP Methods (GET, POST, PUT, PATCH, DELETE)

## Test Scenarios
- Positive and negative test cases
- Status code validation
- Response body validation
- Authorization handling with token
- Data consistency checks

## Project Structure
- Postman Collection
- Environment Variables
- Test scripts using JavaScript (Postman)

## How to Run
1. Import the Postman collection
2. Import the environment file
3. Run the collection via Postman Runner
